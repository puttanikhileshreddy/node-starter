# Node starter Guide 
    >  install node js LTS(Long Term Support)  from https://nodejs.org/en/ 

# Clone the project 

* Open current folder in command prompt and 
    * For sample core Node use node folder 
        * cd node 
        > For runing the project 
         * node> npm start
    * For Express 
        * cd express
        > For runing the project 
         * express> npm install
         * express> npm start
